AiRcruit Interface

I contributed to the Backend Development for this project. As our university did not offer a course on Django, the team learned the framework independently to build the core logic. I led the development of the database architecture; to ensure accuracy, I first built the database in SQL externally before implementing the equivalent logic within the project using Python.

Important: Static UI Preview

This demo is a frontend-only visualization. Certain sections, including profile pages and candidate or company-restricted areas, have been omitted. Because the Django framework and live database are not active in this environment, some pages may not display logically or process data. Form submissions, account registration, and authentication flows are currently non-functional and serve only as a design preview.
